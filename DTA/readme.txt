1. После запустить файл MainTestScript.m на исполнение. Предварительно необходимо указать размеры карты, параметры WORD, количество агентов и целей.
2. В папке "agents" будет создано n файлов с именем agent(ID).mat в которых будет содержаться информация БПЛА (id, type, start, path), где ID - идентификатор БПЛА.
3. Запустить bat файл созданной карты или выбрать готовую из папки USARRunMaps.
4. Запустить на выполнение файл USARSimRun. Если необходимо изменить параметр n.
На карте поочеродно будут появляться БПЛА и их цели. Каждая цель которую настиг беспилотник исчезнет с карты. Как только беспилотник достиг всех целей, он тоже исчезает и появляется следующий. И так до тех пор пока все беспилотники не выполнят свои задания.
В конце рисуется график путей каждого беспилотника с возможным отклонением от маршрута. Возможны незначительные отклонения из-за физических явлений, таких как ветер, препядствия, поэтому в дальнейшем нужно постоянно корректировать маршрут.
